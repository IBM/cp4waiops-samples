#   3) fvAEPg        - A set of requirements for the application-level EPG instance          
#   4) fvEpP         - An endpoint profile
#   5) fvBD          - A bridge domain is a unique layer 2 forwarding domain that contains one or more subnets

#   7) fvSubnet      - A subnet defines the IP address range that can be used within the bridge domain
#   8) vzBrCP        - A contract is a logical container for the subjects which relate to the filters that govern the rules for communication between endpoint groups (EPGs)
#   9) vzSubj        - A subject is a sub-application running behind an endpoint group (for example, an Exchange server). A subject is parented by the contract, which can encapsulate multiple subjects     
#  10) vzOOBBrCP     - An out-of-band binary contract profile can only be provided by an out-of-band endpoint group and can only be consumed by the external prefix set
#  11) vzFilter      - A filter policy is a group of resolvable filter entries
